## 3. 경북독서친구 분석 (bookfriend.info.go.kr) — "결핍 = 기회"

### 3-1. 현재 구조
- 독후활동(권장·일반도서) + **단계 진급·인증·평가**, 한학기한권읽기, 독서동아리, **독서광장(시상·이달의 최다인증 랭킹·이벤트·책추천릴레이·토론광장)**, 나의 독서현황.

### 3-2. 이미 있는 것(중복 주의)
- 독후감 제출, **진급/인증/시상/최다인증 랭킹**, 동아리. → **단순 랭킹은 차별점 약함**.

### 3-3. ★결정적 결핍 (책갈피의 이유)
1. **AI 피드백 전무** — 쓰고 인증할 뿐, 내용 개선·발전 제안 없음(사용자 관찰과 정확히 일치).
2. **손글씨 연계 없음** — 온라인 텍스트 위주, 종이 독후감 OCR 흐름 부재.
3. **질적 성장 추적 부재** — 인증 횟수만, 글의 질(어휘·구성·사고) 성장 미표시.
4. **몰입 보상 루프 약함** — 인증/시상은 있으나 포인트→아바타 즉각보상 약함.
5. **교사 첨삭 부담** — AI 보조로 업무경감(시사성·우수성 근거).

> 포지셔닝: **"경북독서친구가 *읽기를 권장*한다면, 책갈피는 *쓴 글을 길러주고(AI 발전 첨삭)* *손글씨를 디지털로 잇는다(OCR)*."**

### 3-4. 동료 '정보 이음'(xn--on3b62qods0a.com)
- 초등 정보교육 데이터 플랫폼, **AI 누가기록 자동화 + 실시간 수업제어**. 참고: 교사 대시보드 UX·학급 데이터 운영. **출처표기·이용동의 전제로만 참고**(표절/중복 금지).

## 4. 교육과정 연계 (2022 개정, 5~6학년군 국어) ★타당성·독창성 근거

독후감·AI첨삭·공유에 직결되는 성취기준과, AI 첨삭 루브릭이 겨냥할 **성취수준(A/B/C)**:

| 영역 | 성취기준 | 책갈피 연계 | 첨삭 루브릭 타깃(B→A) |
|---|---|---|---|
| 쓰기 | **[6국03-03]** 체험한 일에 대한 **감상을 나타내는 글** | 독후감(감상문) 작성의 본 성취기준 | 감상을 "풍부하게" 드러내도록 확장 |
| 쓰기 | **[6국03-05]** 쓰기 과정 점검·조정, **글 전체 통일성 있게 고쳐 쓰기** | **AI 첨삭→리라이트** 루프의 정당화 | 내용·표현을 통일성 있게 고쳐쓰기 유도 |
| 쓰기 | **[6국03-06]** 쓰기에 적극 참여, **글을 독자와 공유** | 학급 공유·또래 피드백·랭킹 | 다양한 독자와 공유 태도 |
| 읽기 | **[6국02-05]** 긍정적 읽기 동기·적극적 태도 | 포인트·아바타·챌린지 동기설계 | 주도적 읽기 참여 |
| 읽기 | **[6국02-03]** 글 내용의 타당성·표현 적절성 평가 | 비평적 독후감 지도 | 평가 근거 종합 |
| 문학 | **[6국05-06]** 작품을 **자신의 삶과 연관 지어 성찰** | **발전적 첨삭의 사고 확장 타깃** | 삶과 적극 연관·성찰 |
| 문학 | **[6국05-04]** 인상적인 부분 중심 **의견 나누기** | 또래 공유·토론 기능 | 이유를 작품과 연결해 설명 |
| 문학 | **[6국05-03]** 소설/극의 인물·사건·배경 파악 | 독후감 구조화 도우미 | 요소를 적절히 정리 |
| 매체 | **[6국06-03]** 복합양식 매체 자료 제작·공유 | 디지털 독후 산출물 제작·공유 | 수용자 반응 고려 제작 |

> **설계 핵심**: AI 첨삭 출력 = 「칭찬(현재 잘함) / 개선(맞춤법·문장 가볍게, [6국04-06] 연계) / **발전(성취수준 A로 가는 질문·제안, [6국03-03]·[6국05-06] 연계)**」. 즉 *교육과정 성취수준이 AI 첨삭의 채점기준*이 된다 → 일반 챗봇 첨삭과 본질적 차별.

## 5. 책갈피 기능 설계안

### 5-1. 핵심 파이프라인 (차별화의 심장)
```
손글씨 독후감(종이) → 사진 촬영 → 한글 손글씨 OCR → 텍스트 정제
  → AI 첨삭 엔진(서버측 API, 성취수준 루브릭)
     ├─ 칭찬: 현재 글의 강점
     ├─ 개선: 맞춤법·문장 호응(가볍게)
     └─ 발전: 사고 확장 질문·제안(★ A수준 유도)
  → 학생 리라이트 → 성장 기록(포인트·성장그래프) → 학급/학교 공유·랭킹
```

### 5-2. 기능(우선순위)
**MVP(7/31 예선 필수):** 교사 학급개설+학생가입 / 독후감 입력(키보드 + **손글씨 사진 OCR**) / **AI 발전 첨삭 3블록** / 포인트+**아바타 상점** / 교사 대시보드(이력·성장그래프) / **학급·학교 랭킹**.
**핵심 강화(전국용):** 도서 연계(검색·권장도서) / 성장 리포트(어휘 다양성·문장·사고 깊이 시각화) / 또래 공유·댓글(교사 승인) / 챌린지·뱃지 / 가정 연계.
**확장(시사성):** 다문화·느린학습자 쉬운말 모드 / 교사 첨삭 보조(AI 초안→교사 확정, 업무경감).

### 5-3. 사용자 아이디어 평가
| 아이디어 | 평가 | 비고 |
|---|---|---|
| 학교/학급 포인트 랭킹 | ✅ | 경북독서친구도 유사 → **"성장 기반" 랭킹**으로 차별 |
| 독후감→포인트→아바타 | ✅ | 동기 루프 핵심, 다수 수상작 검증 |
| 동료 '정보이음' 참고 | ✅ | 대시보드 UX, 출처표기 전제 |
| (도출) **AI 발전적 첨삭** | ★최우선 | 진짜 차별점, 독창성 20점 |
| (도출) **손글씨 OCR 연계** | ★최우선 | 오프라인↔디지털, 일반화 점수 |
| (도출) **성취수준 루브릭** | ★최우선 | 교육과정 근거, 챗봇과 분리 |

## 6. 기술 스택 권고
- **프런트/배포**: 반응형 **웹(HTML5/CSS/JS)** — 무설치·전기기·다학교 = 일반화 최적.
- **백엔드**: **Firebase**(Auth/Firestore/Storage) — 학급 단위 설계 검증됨(단 구조·UI는 독자적).
- **OCR**: 한글 손글씨 → 상용 API(클로바/구글 비전 등). 정확도·속도가 안정성 점수 직결 → 또박체부터 안정화.
- **AI 첨삭**: **서버측 LLM 호출**(학생 비노출=연령제한 회피). **성취수준 루브릭 프롬프트**로 칭찬/개선/발전 일관 출력.
- 보조: 이 환경 k-skill `library-book-search`(도서), `korean-spell-check`(맞춤법), `korean-humanizer`(피드백 문장 자연화) 활용 가능.

- 골격은 맞춤법 상담소 구조 차용, **문장은 전부 독자 작성**(표절 6어절 규칙).

- 제목안: **「책갈피」 — 손글씨 독후감을 읽고 키워주는 AI 독서 첨삭 플랫폼**.
