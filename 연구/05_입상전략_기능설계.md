# 전국대회 입상 전략 — 기능 → 심사항목 매핑

> 심사 100점: 연구목적 30(독창성20+타당성10) / 구현기술 20(안정성10+우수성10) / 수업활용 20 / **일반화·시사성 30**.
> 입상의 핵심 = ① **독창성**(맞춤법상담소·경북독서친구와 확실히 다름) ② **효과성의 객관적 증거**(B→A 성장 데이터) ③ **일반화**(누구나·전국). 아래 기능은 모두 특정 심사항목 점수를 노린다.

## 이번에 구현(★)하는 기능과 노리는 점수
| 기능 | 노리는 심사항목 | 왜 입상에 유리한가 |
|---|---|---|
| **5축 루브릭 첨삭**(내용·감상·삶연결·구성·맞춤법) | 독창성20·타당성10 | 챗봇식 "잘했어요"가 아니라 **교육과정 성취수준 기반 진단**. 시각화(바)로 설득력 |
| **고쳐쓰기 before→after 향상도** | 효과성·구현우수성 | 입상 보고서의 핵심 = "효과 측정". 학생마다 **점수 상승 데이터 자동 축적** → Raw Data 그대로 증거 |
| **개인 성장 그래프**(성취수준 추이) | 효과성·동기 | 성장의 가시화 = 학생 동기 + 보고서 도표 |
| **뱃지·업적** | 구현우수성(학습동기) | 지속 참여 유도, "동기 유발 기능" 명시 충족 |
| **교사 학급 분석 인사이트**(약점 자동진단+추천활동) | 수업활용·효율성·시사성 | 교사가 데이터로 수업 설계 → "교사-콘텐츠 인터페이스" + 업무경감 |
| **교내 미션/대회 개설**(도서·기간) | 수업활용·일반화 | 학기 단위 운영 시나리오를 시스템이 직접 지원 |
| **자동 표창장 PDF** | 우수성·시사성(업무경감) | 교사 반복업무 자동화 = 일반화 매력 |
| **다문화·느린학습자 쉬운말 모드** | 일반화20·시사성10 | 포용(맞춤법상담소가 다문화로 득점) → 우리는 난이도 적응 피드백 |
| **원고지 모드·글자수** | 독창성(손글씨 연계) | OCR 친화 + 오프라인 쓰기 강조, 차별 포인트 강화 |

## 다음 단계 후보(◐ 추후)
스트릭·독서나무 누적 시각화, 또래 응원/우수작 게시, 독서 토너먼트 투표, 가정 연계 열람코드, 권장도서 연계, 학급 성장 리포트 PDF, 부정·표절 점검.

## 보고서에서 이 기능들을 어떻게 쓸 것인가
- Ⅳ장(연구 결과): **고쳐쓰기 향상도 데이터**(사전 평균 점수 → 사후) + 성취수준 분포 변화 표·그래프(시스템이 자동 산출).
- Ⅲ장(활용): 교사 인사이트·미션·표창장으로 **학기 단위 수업 루프** 제시.
- Ⅴ장(일반화): 쉬운말 모드·웹 무설치·전국 학교 = 누구나 적용.
