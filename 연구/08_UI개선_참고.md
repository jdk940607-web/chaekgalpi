# UI/UX 개선 참고 — 정보이음·독서로·경북독서친구 분석 (다음 세션용)

> 2026-06-29 사용자가 3개 레퍼런스 스크린샷 제공. 책갈피 UI를 "진짜 서비스" 체급으로 올리기 위한 벤치마크.

## 1. 정보이음 (동료교사 사이트) — ★UI 1순위 참고
"보상 및 랭킹 / 명예의 전당" 대시보드가 매우 세련됨. 가져올 것:
- **좌측 사이드바 내비**(접이식): 상단 탭 대신 사이드바. 그룹화 = 수업진행 / 학생관리 / 수업설계 / 시스템마스터. → 교사 화면을 "관리 콘솔" 느낌으로. (학생 화면은 지금 탭 유지해도 OK)
- **명예의 전당 히어로 카드**: 보라 그라데이션 큰 배너 + "학급 전체 포인트 6,279pts · 정보 마스터 1명 · 현재 OOO 1위" 라이브 문구.
- ★**포디움 Top-3 랭킹**: 1·2·3위를 단상 카드로(가운데 1위 크게+👑, 좌2·우3). 지금 책갈피는 리스트형 → 포디움으로 바꾸면 시각적 임팩트 큼.
- **아바타 카드 = 장착 아이템 그리드**(인벤토리/로커 느낌)로 표시.
- **학급 선택 드롭다운 + 마스터 도달률 progress**(예: 4% / 목표 80%).
- **레벨 칭호 chip**: "정보 비기너→정보 마스터" 식. 책갈피는 씨앗→열매 있음 → 칭호 chip 추가.
- 학생 카드마다 **"+지급"**(교사가 포인트 수동 지급) 버튼. teacher에 MASTER 배지.

## 2. 독서로 (read365) — 책 발견(콘텐츠) 섹션 참고
- **가로 캐러셀 책 섹션 다종**: 기관추천도서 / 이달의 추천도서 / 인기 독후활동 도서 / 리뷰가 많은 도서 (각 표지+제목+저자, +더보기).
- **도서 대출 순위 초/중/고 Top 30**(표지+순위 리스트).
- → 책갈피 홈/추천에 "이달의 추천·인기 독후활동·대출 순위" 스타일 **가로 캐러셀**을 우리 스타일화 표지로 추가하면 콘텐츠가 풍성해 보임. (실제 인기도서 예: 고양이 해결사 깜냥, 회색인간, 나는 너를 아는데, 기억 전달자, 아몬드 등 — BOOKS에 보강 가능)
- 메인: AI·빅데이터 맞춤 책 추천 강조 → 우리도 "학년/관심사 맞춤 추천" 카피 활용.

### 제미나이 비전 OCR + 사진 원본 교사 열람(2026-06-29)
- **OCR 업그레이드**: 사진 OCR을 키 있으면 **제미나이 비전(gemini-2.5-flash inlineData)** 우선, 없으면 Tesseract.js 폴백. `geminiOCR(dataUrl)`·`OCR_PROMPT`(원문 그대로·맞춤법 보존) 추가. 한국어 손글씨 정확도 Tesseract보다 크게↑(03_API연동 설계대로 멀티모달 OCR 실현).
- **사진 원본 저장·열람**: `compressImage(file,1000,0.62)`로 다운스케일 JPEG dataURL을 `draft.photo`→`report.photo`로 저장(localStorage). `photoBlock(photo,label)` 헬퍼로 **교사 검토(renderTEdit)·학생 본인 보기(openReport)**에 원본 표시, 검토 리스트(tReview)에 `📷 원본` 태그. OCR된 글 위에 "OCR·입력된 글" 라벨 구분.
- 검증(DOM): 헬퍼 3종 존재, 저장사진 디코드(80×50), 교사 검토뷰 img 로드(naturalWidth>0, data:image/jpeg), 학생뷰·리스트태그 표시, 콘솔에러0. **단 제미나이 OCR 실호출은 키 필요**(프리뷰=별도 브라우저라 사용자 localStorage 접근불가) → 사용자가 앱 로그인화면 BYOK 필드에 키 입력 후 사진 업로드로 실검증 필요.
- ⚠️ localStorage 용량: 압축해도 사진 누적시 한도(~5MB) 주의. 실서비스는 서버/스토리지 업로드로 전환(03_API연동).
