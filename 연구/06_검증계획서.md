# 책갈피 효과 검증 계획서 — 연구보고서 Ⅳ장(연구 결과) 근거

> 목적: "책갈피를 실제 수업에 적용해 **효과가 있었다**"를 객관적 데이터로 입증. (SW·AI분과 심사: 구현 우수성·수업활용·일반화·시사성)
> 원칙: 사전·사후 비교 + 성취수준 기반 루브릭 + 동기/교사 설문. **원자료(Raw Data)는 입상 후 1년 이상 보관**(요강 의무, 미제출=등외).

---

## 1. 검증 설계 개요
| 항목 | 내용 |
|---|---|
| 대상 | 초등 5~6학년 **2개 학급 이상**(공동 2인 각 1학급 권장), N≈40~60 |
| 기간 | **6~8주**(주 1회 독후감 수업 + 가정 연계) |
| 설계 | 단일집단 **사전–사후 비교**(가능 시 미적용 비교반 1개 추가) |
| 도구 | ① 글쓰기 루브릭(성취수준 A/B/C) ② 독서 동기·태도 설문 ③ 교사 설문 ④ 시스템 자동 산출 데이터 |
| 분석 | 사전·사후 평균 차이, 성취수준 분포 변화, 향상도(고쳐쓰기), 만족도 |

---

## 2. 글쓰기 루브릭 (5축 · 각 5점 = 25점) ★핵심 측정도구
시스템 AI첨삭과 동일한 5축 → 교사 채점과 비교·교차검증 가능.

| 영역 | 성취기준 | 5점(A) | 3점(B) | 1점(C) |
|---|---|---|---|---|
| 내용 이해 | [6국05-03] | 인물·사건·배경을 구체적으로 정리 | 핵심 사건을 찾아 씀 | 줄거리 일부만 |
| 감상 표현 | [6국03-03] | 느낌을 풍부하고 생생하게 | 느낌을 나타냄 | 느낌이 거의 없음 |
| 삶과 연결 | [6국05-06] | 경험+다짐으로 적극 성찰 | 경험을 떠올려 연결 | 연결 없음 |
| 구성·근거 | [6국03-05] | 까닭·연결어로 짜임새 있게 | 까닭을 일부 제시 | 나열식 |
| 맞춤법 | [6국04-06] | 오류 거의 없음 | 일부 오류 | 오류 잦음 |

- 채점: 담임 + (가능 시) 동료 교사 2인 교차 채점, 불일치 시 협의.
- 사전: 적용 전 자유 독후감 1편 / 사후: 8주차 독후감 1편(동일 루브릭).

## 3. 동기·태도 설문 (학생, 5점 척도, 사전·사후) — 「맞춤법 상담소」 양식 참고·재작성
- 나는 독서가 재미있다 / 책을 읽고 글 쓰는 것이 좋다 / 스스로 책을 찾아 읽는다
- 내 독후감이 좋아지고 있다고 느낀다 / AI 피드백이 글쓰기에 도움이 된다
- 친구의 글을 읽고 이야기 나누는 것이 즐겁다 / 다음에도 책갈피로 쓰고 싶다
- 자유서술: 좋았던 점 / 더 바라는 점

## 4. 교사 설문 (5점 척도)
- 채점·기록 업무 부담이 줄었다(업무경감) / 학생 성취수준 파악이 쉬웠다
- 개별 맞춤 지도에 도움이 되었다 / 학생 독서 동기가 높아졌다
- 다른 교사에게 추천하겠다 / 인터페이스 사용성

## 5. 시스템 자동 산출 데이터 (책갈피가 자동 수집 = 강력한 증거)
- 학생별 **성취수준 추이**(성장 그래프), **고쳐쓰기 향상도**(평균 점수 ▲)
- 학급 5축 루브릭 평균·약점 영역(인사이트), 제출 편수·참여율
- **CSV 내보내기**로 원자료 그대로 확보 → Raw Data 보관·제출 대응
- 게임·토론 참여 등 동기 지표

## 6. 기대 결과(가설)
1. 사후 글쓰기 루브릭 평균 **유의하게 상승**(특히 삶연결·감상 표현).
2. 성취수준 **C→B→A 이동**, A비율 증가.
3. 고쳐쓰기 1회로 평균 점수 **+1점 내외 향상**(즉각 피드백 효과).
4. 학생 동기·교사 업무경감 설문 **4점 이상**.

## 7. 일반화·시사성 연결(보고서 Ⅴ장)
- 웹·무설치·전국 6,300여 학교 적용 가능 → 일반화.
- 서버측 AI로 연령제한 회피·개인정보 안전 → 정책적 시사.
- 경북독서친구의 "권장·인증" 한계를 **발전적 피드백·성장 추적**으로 보완 → 차별·시사.

## 8. 윤리·규정
- 학생/보호자 **개인정보 수집·이용 동의서**(서식9), 사진·독후감 외부 비전송.
- 생성형 AI는 **서버측 호출**(학생 직접 사용 금지=연령제한 준수).
- 설문 원자료·결과물 **익명화**, 보관기간 후 파기.
