# 📖 책갈피 — 읽고, 쓰고, 자라는 AI 독서 친구

초등학생의 독후감 쓰기를 **손글씨 OCR + AI 첨삭 + 과정중심평가**로 지원하는 학교 독서교육 플랫폼입니다.

## 1. 무엇을 하는 서비스인가

- 학생이 독후감을 **키보드·원고지·손글씨 사진(OCR)** 3가지 방식으로 제출
- AI가 2022 개정 국어과 성취기준 기반 **5축 루브릭**(내용이해·감상표현·삶과연결·구성근거·맞춤법)으로 즉시 첨삭
- 학생은 피드백을 보고 **고쳐쓰기** → 전후 비교(Diff)와 향상도가 자동 기록 (과정중심평가)
- 교사는 AI 초안을 검토·수정해 **최종 확정** (AI 초안 → 교사 확정 이중 구조)
- 게임·아바타·랭킹으로 독서 동기 부여, 생기부 자동 요약까지 연결

## 3. 핵심 기능 상세

### ✍️ 독후감 쓰기
- **손글씨 OCR**: 공책에 쓴 독후감을 사진으로 찍으면 Gemini Vision이 텍스트로 변환. 원본 사진은 교사 검토 화면에서도 열람 가능
- **AI 5축 첨삭**: 성취기준 코드([6국05-03] 등)와 연결된 루브릭 점수 + 칭찬/개선/확장 피드백
- **맞춤법 검사**: Daum 검사기 서버 프록시 (실패 시 내장 사전 폴백)
- **음성 입력(STT)·읽어주기(TTS)·쉬운 말 모드**: UDL(보편적 학습 설계) 지원
- **고쳐쓰기 Diff**: 단어 단위 전후 비교(추가=초록/삭제=빨강) + 성취수준 향상도 자동 산출

### 📚 책 발견 (도서관 정보나루 실데이터 연동)
- **전국 초등 인기대출 Top 10**: 8~13세 실시간 대출 순위
- **연계 추천**: 책을 열면 "함께 대출된 책 + 마니아/다독자 추천" — AI가 아닌 **전국 대출 빅데이터 기반**
- **책 탐험 체인**: 추천 책을 누르면 그 책의 모달이 또 열려 꼬리에 꼬리를 무는 탐색
- **우리 동네 도서관에서 빌리기**: 학교 소재 지역의 공공도서관 소장 여부 + **지금 대출 가능한지 실시간 확인**
- **키워드로 책 찾기**: 키워드 칩 클릭 → 그 주제 책 검색
- **나의 독서 키워드 지도**: 독후감을 낼 때마다 그 책의 키워드가 쌓여 취향을 시각화 (메타인지)

### 🎮 독서 게임 8종
골든벨 · 밸런스게임 · 빙고 · 책 퀴즈 · 퀴즈 대결 · 어휘게임 · 인물맞히기 · 마라톤 + **고전 퀴즈**(저작권 만료 작품 — 흥부전·심청전·별주부전·소나기 등)

### ❓ 책 퀴즈 3중 저작권-안전 설계
1. **학생 출제형**: 독후감 제출 후 "친구를 위한 퀴즈 만들기" — 책을 읽은 학생이 직접 출제(동료학습, 저작권 무결)
2. **교사 AI 초안 + 검수**: 출판사 공식 책 소개를 근거로 AI가 초안 생성 → 교사가 수정 후 출제
3. **고전 퀴즈**: 저작권 만료 작품만 깊이 있는 내용 문제

### 🧸 아바타 상점
- SVG 코드로 그린 커스텀 캐릭터(5등신 일러스트) — 헤어·모자·상의·하의·신발·액세서리 6슬롯
- 아이템 단독 프리뷰 카드, 희귀도 등급(레어/레전드), 구매 컨페티, 숨쉬기·눈깜빡임 애니메이션
- 독후감으로 모은 포인트로 구매 (독서 → 보상 선순환)

## 4. 기술 구조

```
프론트엔드  바닐라 JS 단일 HTML (~680KB, 프레임워크 없음)
저장소      localStorage (DB_VERSION 3) — 서버 DB 전환 설계 완료(Firebase)
서버        Vercel Serverless Functions (무료)
  /api/gemini         Gemini 프록시 (운영자 키 서버 보관 — 학생 키 입력 불필요)
  /api/spell          Daum 맞춤법 프록시
  /api/booksearch     카카오→네이버 책 검색 (표지·소개)
  /api/popular-books  정보나루 인기대출도서
  /api/library        정보나루 연계추천·트렌드·도서관 소장 확인
배포        GitHub push → Vercel 자동 배포
```

**연동 외부 API**: Google Gemini · 카카오 책검색 · 네이버 책검색 · Daum 맞춤법 · **도서관 정보나루(국립중앙도서관 공공데이터)** · NEIS 전국 초등 6,331개교 실데이터(로그인 학교 검색)

## 5. 설계 원칙

- **AI 초안 → 교사 확정**: AI는 보조, 최종 평가권은 교사에게 (생성형 AI 연령 제한 약관 리스크도 서버 프록시로 회피 — 학생이 모델을 직접 호출하지 않음)
- **저작권 준수**: 책 원문을 수집·저장하지 않음. 출판사 공식 소개 데이터(오픈 API)와 학생 직접 출제만 활용
- **투명한 구현 구분**: 실구현(✅)/시연용(⚠️)/향후과제(❌)를 보고서에 명시. 학교 도서관(DLS)은 교육청 폐쇄망이라 현재 시연용 — DLS API 개방 시 공공(정보나루)+학교 이중 데이터 체계로 확장 예정
- **접근성(UDL)**: 음성 입력·읽어주기·쉬운 말 모드로 다문화·느린 학습자 지원
