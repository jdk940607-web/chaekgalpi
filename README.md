# 책갈피 — AI 독서 첨삭 플랫폼 (정보화연구대회 출품 프로젝트)

손글씨 독후감을 OCR로 읽어, AI가 **내용·사고력**을 발전적으로 첨삭해 주는 초등 독서활동 플랫폼.
제20회 디지털교육연구대회 **교육용 SW·AI분과** 출품 준비.

## 폴더 구성

## 프로토타입 실행
```
prototype/index.html 을 크롬으로 열기 (schools.js 같은 폴더 필요)
```
데모 로그인: '빠른 데모 입장' → 전동규 선생님(경산동부초 5-1) 또는 학생들. 모든 데모 비번 = 1234.
실제 로그인: 학교 검색 → 학년/반 → 이름 → 비번. 비번 분실 시 교사가 '학급·학생 관리'에서 초기화(1234).

## 핵심 차별점 (요약)
1. 맞춤법이 아닌 **독후감 내용·사고의 발전적 첨삭** (기존 1등급 '맞춤법 상담소'와 분리)
2. **손글씨 OCR**로 오프라인 독서를 디지털 연계
3. AI 첨삭 루브릭 = **2022 개정 국어 성취수준(A/B/C)** 매핑 → 일반 챗봇과 차별
4. 경북독서친구의 "권장·인증" 결핍을 **발전적 피드백 + 성장 추적**으로 보완
